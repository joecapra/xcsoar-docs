# 8 Airspace, Traffic and Team Flying

:::note
This chapter was not present in the extracted PDF source. Please refer to the official XCSoar documentation at [xcsoar.org](https://xcsoar.org) for information on airspace management, FLARM traffic awareness, and team flying features.
:::

## 8.1 Airspace

XCSoar can display Special Use Airspace (SUA) and warns pilots when they are predicted to enter or are near SUA boundaries. Airspace data must be provided in OpenAir or TNP format data files.

## 8.2 Airspace Display

Airspace areas are displayed on the map as shaded regions with thick borders. The color and pattern of the regions reflect the airspace class.

## 8.3 Airspace Warnings

XCSoar constantly checks whether the aircraft's predicted track will pass through any airspace. When airspace is about to be penetrated, an audible warning is given and a dialog box pops up showing the airspace that is about to be violated.

## 8.4 FLARM Traffic

FLARM is an electronic anti-collision system widely used in gliders. When connected to a FLARM device, XCSoar can display the positions of other FLARM-equipped aircraft on the map.

## 8.5 Team Flying

XCSoar supports team flying through the Team Code feature. Team code is a compact encoding of a relative position that can be communicated over radio so team members can locate each other.
